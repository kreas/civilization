# Homepage Landing Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Civilization Tech marketing homepage following the "Architectural Monolith" design system, and move the existing auth page to `/login`.

**Architecture:** Six presentational components (`navbar`, `hero`, `services`, `selected-works`, `cta`, `footer`) composed in a new `src/app/page.tsx`. Existing auth page moves to `src/app/login/page.tsx`. Fonts loaded via Adobe Typekit (Aktiv Grotesk) and next/font/google (Inter). Design system colors added to `globals.css` alongside existing ShadCN variables.

**Tech Stack:** Next.js 16, Tailwind CSS v4, Adobe Fonts (Typekit), Vitest + React Testing Library

**Spec:** `docs/superpowers/specs/2026-03-30-homepage-design.md`
**Design reference:** `sources/inspiration/homepage-design.png` and `sources/inspiration/homepage-design.html`

- [ ] **Step 1: Create the login page**

Create `src/app/login/page.tsx` with the existing auth page content:

```tsx
import { withAuth, getSignInUrl, getSignUpUrl } from '@workos-inc/authkit-nextjs';

export default async function LoginPage() {
  const { user } = await withAuth();
  const signInUrl = await getSignInUrl();
  const signUpUrl = await getSignUpUrl();

  return (
    <div className="flex min-h-screen items-center justify-center">
      <main className="flex flex-col items-center gap-8 p-8">
        <h1 className="text-4xl font-bold tracking-tight">Civilization Tech</h1>

        {user ? (
          <div className="flex flex-col items-center gap-4">
            <p className="text-lg text-muted-foreground">
              Signed in as <span className="font-medium text-foreground">{user.email}</span>
            </p>
          </div>
        ) : (
          <div className="flex flex-col items-center gap-4">
            <p className="text-lg text-muted-foreground">Get started by signing in.</p>
            <div className="flex gap-3">
              <a
                href={signInUrl}
                className="inline-flex h-10 items-center justify-center rounded-md bg-primary px-6 text-sm font-medium text-primary-foreground transition-colors hover:bg-primary/90"
              >
                Sign In
              </a>
              <a
                href={signUpUrl}
                className="inline-flex h-10 items-center justify-center rounded-md border border-input bg-background px-6 text-sm font-medium transition-colors hover:bg-accent hover:text-accent-foreground"
              >
                Sign Up
              </a>
            </div>
          </div>
        )}
      </main>
    </div>
  );
}
```
